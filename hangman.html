Hangman

First app that I made in Android Studio. I wanted to learn the process behind getting an app on the Google Play store. Also, I wanted to learn how to run ads in an application. Using AdMob and the google play store I was able to learn a lot about the development services offered. Currently, I am also looking to add logging and crash reports with firebase.

Frontend: XML

Backend: JAVA

Database: SQLite

  • Designed, Built, Tested, and Deployed an application by myself
  • Implemented banner ads in an application
  • Created an App that is only 5MB and 4MBs of that is just for Google ads
